# ANVESHAK - OCR Based Intelligent PDF Search System

## Overview
ANVESHAK is a Python-based system that extracts and searches text from PDF documents using OCR and native parsing techniques.

## Features
- OCR using Tesseract
- PDF text extraction
- Indexed search with highlights
- Metadata extraction
- Multithreading for faster processing

## Tech Stack
- Python
- Flask
- Tesseract OCR

## Usage
Upload a PDF and search for keywords within the document.
